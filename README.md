# ios_medioa_converter

This is a converter for ios media files. It can convert to jpg, mp4 and so on.

## Requirements
- pyheif(available on Linux and MacOS)
- tqdm
- moviepy

## Installation

```bash
pip install pyheif tqdm moviepy
```

## Usage
Please Input ur file path in the code.
`ios2compatible.py`

```python
DIR_PATH = "input ur directory path here"
```
Then run the code.


```bash
python3 ios2compatible.py
```
